# Justice for Bangladesh 2024 Website

### How to build on local
- Make `local_serve.sh` an executable. E.g. `chmod a+x local_serve.sh`.
- `./local_serve.sh build`
- `./local_serve.sh serve`
- Open `http://localhost:4000` in your browser.
